# travel
2022年春哈工大（威海）web开发技术与实践大作业 ，旅游网后端实现

**3.1. Web** **层** 

a) Servlet：前端控制器

b) html：视图 

c) Filter：过滤器

d) BeanUtils：数据封装 

e) Jackson：json 序列化工具 

**3.2. Service** **层** 

f) Redis：nosql 内存数据库 

g) Jedis：java 的 redis 客户端 

**3.3. Dao** **层** 

h) Mysql：数据库 

i) Druid：数据库连接池

j) JdbcTemplate：jdbc 的工具
